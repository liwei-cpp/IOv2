template<io_converter KernelType>
template<bool Saturate = false>
auto IOv2::cvt_reader< KernelType >::get_buf ( size_t  to_max)
inline

Retrieves data from root_cvt's internal buffer, reading from device if needed.

模板参数
SaturateControls behavior when insufficient data is available:
  • false (default): Returns available data (may be less than requested). Return type: std::pair<const char_type*, size_t>
  • true: Requires exactly to_max chars; throws if not enough data. Return type: const char_type*
参数
to_maxNumber of characters to retrieve.
返回
When Saturate=false: pair of (pointer, actual_count). When Saturate=true: pointer to exactly to_max chars.
异常
cvt_errorIf Saturate=true and end-of-stream is reached before reading to_max characters.
