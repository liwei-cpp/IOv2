Detailed Description

template<typename TK, typename TV, size_t Capacity>
requires std::is_copy_constructible_v<TK> && std::is_copy_constructible_v<TV> && std::equality_comparable<TK> && requires { std::hash<TK>{}; }
class IOv2::lru_cache< TK, TV, Capacity >

A simple Least Recently Used (LRU) cache.

When the cache reaches capacity, the least recently accessed item is evicted. Both get() and put() operations have O(1) average time complexity.

Template Parameters
TKKey type, must be copyable, equality comparable, and hashable
TVValue type, must be copyable
CapacityMaximum number of items to store
Note
This class is NOT thread-safe.
Example
lru_cache<std::string, int, 100> cache;
cache.put("key1", 42);
if (auto val = cache.get("key1")) {
// For small types, val is std::optional<int>
std::cout << *val << std::endl;
}

return_type IOv2::lru_cache< TK, TV, Capacity >::get ( key_param_type  key)
inline

Get a value from the cache.

If the key exists, the item is marked as recently used (moved to front).

Parameters
keyThe key to look up
Returns
std::optional<TV> for small types, const TV* for large types; returns empty optional or nullptr if key not found
Warning
For large types (is_small_type_v<TV> == false), the returned pointer points to internal storage. Subsequent calls to put() or try_put() may invalidate this pointer (when the cache evicts this item). If you need to hold the value long-term, copy it immediately:
if (auto* p = cache.get(key)) {
TV copy = *p; // Copy immediately
// Now safe to call other cache operations
}

Put a key-value pair into the cache (updates existing values).

If the key already exists, updates its value and marks it as recently used. If the key doesn't exist, inserts a new item; evicts the LRU item if cache is full.

Try to put a key-value pair into the cache (without updating existing values).

If the key already exists, only marks it as recently used without updating the value. If the key doesn't exist, inserts a new item; evicts the LRU item if cache is full.
