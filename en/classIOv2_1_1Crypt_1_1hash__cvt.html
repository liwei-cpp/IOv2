Detailed Description

class IOv2::Crypt::hash_cvt< KernelType, TInt >

Write-only hash-digest converter.

Accepts data written by the user and continuously updates the internal hash state via put_main(). The digest is written to the underlying kernel at the following points:

  • On detach() (or object destruction) if there is unwritten main content.
  • When manually triggered via adjust(dump_hash{}).

The default output format is lowercase hexadecimal (hash_fmt::lower_hex), switchable via adjust(set_hash_fmt{...}). Only write (output) mode is supported; get is not available.

Template Parameters
KernelTypeThe underlying IO converter kernel type; must satisfy io_converter and support put.
TIntThe user-facing element type (internal_type), defaulting to the kernel's internal_type. Must be a single-byte, trivially copyable type with unique object representations.
