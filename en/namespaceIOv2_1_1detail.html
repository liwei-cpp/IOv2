Convert a locale-encoded narrow string to UTF-32.

Parameters
valInput bytes interpreted under locale_name.
locale_nameLocale name passed to the underlying codecvt_kernel.
Returns
Decoded UTF-32 string.
Exceptions
cvt_errorIf val ends in the middle of a multi-byte sequence (partial trailing bytes), or contains an invalid sequence under the locale. This is NOT a best-effort conversion — callers must pass a complete byte stream.

Convert a UTF-8 string to UTF-32.

Parameters
valInput UTF-8 bytes.
Returns
Decoded UTF-32 string.
Exceptions
cvt_errorIf val ends in the middle of a UTF-8 sequence (partial trailing bytes), or contains any invalid UTF-8 (overlong, surrogate, out-of-range, malformed continuation). This is NOT a best-effort conversion — callers must pass a complete, well-formed UTF-8 stream.

Convert a UTF-32 string to UTF-8.

Parameters
valInput string. Every element must be a Unicode Scalar Value: a code point in [0, 0x10FFFF] and outside the surrogate range [0xD800, 0xDFFF]. std::u32string does not enforce this — callers are responsible for sanitizing input.
Returns

Exceptions
