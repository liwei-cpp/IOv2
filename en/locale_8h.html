Detailed Description

Defines the locale class template – a localization object that manages a set of facets. locale is a value-semantic, copy-on-write type whose authoritative state is the per-LC_* base facet confs; it lazily builds composite facets (e.g. numeric, timeio) on demand. It also handles message translation (involve_msg) and lookup of the initial locale names resolved from the environment at program startup.
