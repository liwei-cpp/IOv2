Detailed Description

Defines the specializations of the numeric_conf class template, which are the underlying implementations for the numeric<CharT> facet. Each specialization extracts locale-dependent numeric formatting parameters, including the decimal point character, thousands separator, digit grouping rules, and textual representations of boolean values.
