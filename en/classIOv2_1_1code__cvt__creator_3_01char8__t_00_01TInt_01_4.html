Detailed Description

template<typename TInt>
requires std::is_same_v<TInt, char32_t> || (std::is_same_v<TInt, wchar_t> && wchar_t_is_utf32)
class IOv2::code_cvt_creator< char8_t, TInt >

Factory class for UTF-8 code_cvt instances (char8_t <-> char32_t/wchar_t). Stateless; no locale is required. Creates code_cvt instances for a given underlying kernel via the create() method. Satisfies the cvt_creator concept.
