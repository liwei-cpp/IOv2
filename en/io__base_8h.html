Detailed Description

Defines the infrastructure for the I/O stream hierarchy, including:

  • ios_defs: definitions of the formatting flags (fmtflags) and stream state bits (iostate).
  • io_state_and_exp: manages the stream state bits together with their associated exceptions (exception mask / exception pointers).
  • ios_base: the base class holding the formatting state of every stream: format flags, precision, width, fill character, pword storage, and locale-change callbacks.
  • A set of manipulators that modify a stream's formatting, such as hex, left, fixed, etc.
  • sync: an RAII helper that locks a stream's I/O mutex.
