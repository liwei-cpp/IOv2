Detailed Description

template<io_converter KernelType>
requires (std::is_base_of_v<root_cvt<typename KernelType::device_type, true>, KernelType> && !is_mem_device<typename KernelType::device_type>)
class IOv2::cvt_reader< KernelType >

cvt_reader specialization for root_cvt with an internal buffer (HasInBuffer = true, non-mem_device). Directly reuses root_cvt's internal buffer to avoid extra data copies.

Template Parameters
KernelTypeMust derive from root_cvt<DeviceType, true> with a non-mem_device DeviceType.

cvt_reader specialization for mem_device (applies to both rb_root_cvt and no_rb_root_cvt kernels). mem_device directly exposes its internal buffer pointers, so all operations forward directly to the device without an additional buffering layer.

Template Parameters
KernelTypeMust derive from root_cvt<DeviceType, true|false> with a mem_device DeviceType.

auto IOv2::cvt_reader< KernelType >::get_buf ( size_t  to_max)
inline

Retrieves data from root_cvt's internal buffer, reading from device if needed.

Template Parameters
SaturateControls behavior when insufficient data is available:
  • false (default): Returns available data (may be less than requested). Return type: std::pair<const char_type*, size_t>.
  • true: Requires exactly to_max chars; throws if not enough data. Return type: const char_type*.
Parameters
to_maxNumber of characters to retrieve; must not exceed s_buffer_length.
Returns
When Saturate=false: pair of (pointer, actual_count). When Saturate=true: pointer to exactly to_max chars.
Exceptions
cvt_error
  • When to_max is 0.
  • When to_max exceeds s_buffer_length.
  • When io_status is not input and cannot be switched.
  • When Saturate=true and end-of-stream is reached before reading to_max characters.
