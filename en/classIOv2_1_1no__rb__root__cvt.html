Detailed Description

template<io_device DeviceType>
class IOv2::no_rb_root_cvt< DeviceType >

Convenience alias for root_cvt with the internal read buffer disabled (HasInBuffer = false). Each get() call is forwarded directly to the device without maintaining a read buffer in the converter layer. Use this when rollback is not needed or the device already buffers.

Template Parameters
DeviceTypeThe underlying I/O device type, which must satisfy the io_device concept.
