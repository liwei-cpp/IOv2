Detailed Description

Defines the collate<CharT> facet class, providing locale-aware string comparison and collation-key transformation. The class wraps a shared instance of collate_conf<CharT> and exposes multiple overloads that accept both raw-pointer ranges and generic-iterator ranges as input and output.
