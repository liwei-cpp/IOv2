Botan::secure_vector< uint8_t > IOv2::Crypt::chacha20_cvt_helpers::key_gen ( std::string_view  key)
inline

Derives a ChaCha20 key from an arbitrary passphrase string by hashing it with SHA-256.

Empty input is rejected: hashing an empty string yields the publicly known constant SHA-256(""), which provides zero confidentiality when used as a stream-cipher key.

Parameters
keyThe raw passphrase string; must not be empty.
Returns
A 32-byte SHA-256 digest suitable for direct use as a ChaCha20 key.
Exceptions
cvt_errorIf key is empty, or if the SHA-256 hash object cannot be created.
