Detailed Description

Defines the implementation details of the messages facet, including the base_ft<messages> base class and per-character-type specializations of messages_conf. base_ft<messages> manages gettext text-domain binding, language resolution, and loading and parsing of .mo files; the messages_conf specializations build per-character-type translation dictionaries on top of that base.
