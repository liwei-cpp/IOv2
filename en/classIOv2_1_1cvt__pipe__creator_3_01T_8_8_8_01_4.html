Detailed Description

template<typename... T>
requires (sizeof...(T) > 2)
class IOv2::cvt_pipe_creator< T... >

Multi-element pipeline creator (variadic specialization, requires more than two elements). Internally, all creators are stored flattened in a single std::tuple. When create(kernel) is called, it wraps kernel with each creator in index order, yielding a multi-layer nested converter. The outermost layer corresponds to the last creator (highest index), and the innermost layer (directly wrapping kernel) corresponds to the first creator (index 0).

Template Parameters
T组成管道的各转换器工厂类型（至少三个）。 / The converter creator types forming the pipeline (at least three).
