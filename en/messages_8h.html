Detailed Description

Defines the messages<CharT> class, the user-facing facet for gettext-style message translation. It wraps a messages_conf<CharT> instance and provides three translate overloads distinguished by value category (lvalue / rvalue / const rvalue), minimizing unnecessary copies while preventing dangling references.
