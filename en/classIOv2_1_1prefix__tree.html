Detailed Description

template<typename CharT, typename TValue>
class IOv2::prefix_tree< CharT, TValue >

Prefix tree (Trie) implementation for internal IOv2 components.

This class implements a prefix tree data structure for efficient string prefix matching. Supports longest prefix matching, useful for parsers and similar scenarios.

Template Parameters
CharTCharacter type, must be equality comparable and hashable
TValueValue type associated with strings, must be equality comparable
Note
This class is intended for internal use and does not provide a comprehensive set of public interfaces (e.g., remove, clear, size).
This class is NOT thread-safe.
Example
prefix_tree<char, int> tree;
tree.add("hello", 1);
tree.add("help", 2);
std::string input = "helper";
prefix_tree<char, int>::match_out_type result;
auto end_it = tree.max_match(input.begin(), input.end(), result);
// result == 2 ("help" is the longest match)
// end_it points to "er"
