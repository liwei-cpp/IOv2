Detailed Description

Defines the implementation details of the monetary facet, including the base_ft<monetary> base class and per-character-type specializations of monetary_conf. base_ft<monetary> provides pattern-construction utilities driven by POSIX lconv flags; the monetary_conf specializations read lconv from a locale name and populate all fields required for monetary formatting.
