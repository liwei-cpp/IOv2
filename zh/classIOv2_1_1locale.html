◆ initial_locale_name()

template<typename TChar >
static const std::string & IOv2::locale< TChar >::initial_locale_name ( int  category)
inlinestatic

返回某个 LC_* 类别在**程序启动时由环境变量解析得到**的初始 locale 名称。

这是对 ori_facet_buf::locale_name 的薄转发，使得仅依赖 <locale/locale.h> 的代码（如 stdio 流对象）无需直接看到 ori_facet_buf 即可获取该初始名称。 返回值与字符类型无关；之所以作为 locale 的静态成员暴露，是为了让调用方只 看见 locale。

异常
stream_error由底层 ori_facet_buf::locale_name 透出：当 category 不是已解析的五个类别之一（LC_CTYPE / LC_COLLATE / LC_MONETARY / LC_NUMERIC / LC_TIME）时抛出，例如传入 LC_ALL 或 LC_MESSAGES。
